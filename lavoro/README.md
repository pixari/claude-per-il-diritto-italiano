# lavoro

Plugin per HR, legale interno e studi: assunzioni, disciplina, licenziamenti, giudizio del lavoro. Output sempre **bozza**.

## Per chi è

| Ruolo | Workflow |
| --- | --- |
| Legale HR / interno | Hiring, disciplina, licenziamento |
| Studio contenzioso | Revisione licenziamento, fascicolo |
| HR operativo | Q&A rapida, checklist |

## Prima esecuzione

```
/lavoro:cold-start-lavoro
```
