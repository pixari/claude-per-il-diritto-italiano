# Citazione italiana

## Obbligatorio

- Mai inventare estremi di sentenza.
- Senza accesso a fonte verificata dall'utente: marca **`[da verificare]`**.

## Norme

Formato tipo:

`art. X, comma Y, [l. / d.lgs. / d.P.R.] n. … del …`

- Indica **versione vigente** alla data del fatto o del quesito.
- Segnala rischio **novella** / multivigenza se pertinente.

## Unione europea

- Regolamento UE con numero e data.
- Direttiva: stato recepimento in Italia se rilevante.
- CGUE/CURIA: estremi completi + `[da verificare]`.

## Giurisprudenza

| Organo | Formato |
| --- | --- |
| Cass. | Cass. civ./pen./lav., Sez., sent./ord., data, n. |
| Sez. un. | Indicare esplicitamente |
| Merito | Trib./Corte app., estremi completi |
| Corte cost. | sent./ord., data, n. |
| Amministrativo (v2) | TAR, Cons. Stato — quando applicabile |

## Prassi

Circolari AE, linee Garante privacy, orientamenti CNF: come orientamento, non come legge.

## Checklist prima di consegnare bozza

- [ ] Ogni citazione normativa ha estremi e vigore?
- [ ] Ogni sentenza citata è verificata o `[da verificare]`?
- [ ] Nessun testo di legge intero copiato dalla memoria del modello?
