# Checklist clausole privacy — contratto

> **Bozza — non consulenza legale.** Verificare adeguamento e TIA/transfer tool.

## Ruoli

Identifica: titolare, responsabile, sub-responsabile, joint controller (art. 26 se applicabile) — **bozza**.

## Checklist

| Tema | Verifica |
| --- | --- |
| Definizioni | Allineamento art. 4 GDPR |
| Finalità e istruzioni | Art. 28(3) — documentate |
| Sub-responsabile | Autorizzazione generale/specifica, flusso informativo |
| Misure art. 32 | Riferimento standard contrattuale o allegato TOM |
| Data breach | Notifica titolare, tempi |
| Assistenza | DPIA, diritti interessati, consultazione Garante |
| Cancellazione/restituzione | Fine contratto |
| Audit | Diritti ispettivi proporzionati |
| Trasferimenti extra-SEE | Clausole SCC / decisioni adeguatezza / TIA `[da verificare]` |
| Legge applicabile / foro | Coerenza con resto contratto |

## Output

- Tabella gap + suggerimento formulazione **bozza**
- Flag "richiede DPO / transfer assessment"

Non sostituisce DPA firmato né negoziazione con controparte.
